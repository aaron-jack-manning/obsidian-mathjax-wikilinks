# Obsidian MathJax WikiLinks

Obsidian plugin to render MathJax within WikiLinks.

## Example

The WikiLink `[[sigma-algebra|$\sigma$-algebra]]` renders in Obsidian without rendering the MathJax expression from the alias, whereas `[$\sigma$-algebra](sigma-algebra.md)` will render correctly. This plugin unifies the two versions, rendering MathJax properly in both cases.

## Installation

To install this plugin simply copy `main.js` and `manifest.json` into a folder `VAULT_PATH/.obsidian/plugins/obsidian-mathjax-wikilinks`.
